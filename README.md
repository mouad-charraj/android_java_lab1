# Android Java Lab 1 :

## Description
Ce projet est le premier laboratoire pratique du module de développement mobile Android. Il s'agit d'une application d'initiation permettant de manipuler les composants de base (Widgets), de gérer les événements de clic et de comprendre l'interaction entre le code Java et l'interface XML.

## Fonctionnalités
L'application propose une interface interactive avec les éléments suivants :
* **Compteur Dynamique** : Un `TextView` central qui s'incrémente à chaque interaction.
* **Action "Dire Bonjour"** : Un bouton vert pour déclencher une salutation ou un message.
* **Action "Compter"** : Un bouton bleu dédié à l'incrémentation du compteur.
* **Design Personnalisé** : Utilisation de formes arrondies (rounded corners) et de couleurs spécifiques pour les boutons.

## Technologies Utilisées
* **Langage** : Java
* **IDE** : Android Studio
* **Layout** : XML (ConstraintLayout)
* **Build System** : Gradle
